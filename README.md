# React Login and Signup Form

## Technologies Used
-React
-Formik
-React Router DOM
-TypeScript

## Getting Started

### Installation

1. Clone the repository:
   bash
   git clone
   https://github.com/ritiknamdev07/React-Auth-Form.git

   cd react-auth-form


1. Install Dependencies:
   bash
   npm install

2. Start the Development Server:
   bash
   npm run dev
